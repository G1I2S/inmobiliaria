# Inmobiliaria ADL

Sitio web para una inmobiliaria con listado de propiedades en venta y alquiler.

## Demo

[Ver sitio en vivo](https://g1i2s.github.io/inmobiliaria/)

## Páginas

- **Inicio** — Presentación, estadísticas y propiedades destacadas
- **En Venta** — Listado de propiedades disponibles para comprar
- **En Alquiler** — Listado de propiedades disponibles para alquilar

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)
